💄 Nykaa Frontend Clone — Beauty E-Commerce UI

A fully responsive beauty & skincare shopping web UI inspired by Nykaa.
Built using HTML, CSS, and Vanilla JavaScript with real e-commerce interactions like slider, product listing, live search, wishlist, and cart sidebar.

🎯 Goal: Practice DOM manipulation, UI/UX, and shopping cart logic without any framework.

🧠 What You’ll Learn From This Project
DOM rendering from JavaScript arrays
Event delegation & dynamic UI updates
Cart state management (add/remove/qty/total)
Search & filter logic
Responsive layout with Flexbox & Grid
Clean folder structure for static projects
✨ Features Breakdown
Feature	Description
🖼️ Hero Slider	Auto-play slider with dots, arrows, overlay text, CTA
🛍️ Product Grid	Dynamic cards with brand, rating, price, discount
❤️ Wishlist	Toggle heart icon with toast feedback
➕ Add to Bag	Cart system with quantity management
🧾 Cart Sidebar	Slide-in cart, remove items, live total
🔎 Live Search	Filter products by name or brand in real-time
🧭 Sticky Navbar	Changes style on scroll
🧪 Category Filter	Skincare / Makeup filtering tabs
🍞 Toast Messages	UX feedback for actions
📱 Responsive	Works across desktop, tablet, mobile
🧱 Tech Stack
🌐 HTML5 (semantic structure)
🎨 CSS3 (Flexbox, Grid, transitions)
⚙️ Vanilla JavaScript (ES6, DOM APIs)

No libraries. No frameworks. Pure fundamentals.
