💄 Nykaa Frontend Clone

A responsive beauty & skincare shopping webpage inspired by Nykaa.
Built using HTML, CSS, and JavaScript with features like slider, product cards, add to cart, wishlist, live search, and cart sidebar.

✨ Features
🖼️ Hero Image Slider (Auto + Manual)
🛍️ Product Cards with ratings, price, discount
➕ Add to Bag (Cart system with quantity)
❤️ Wishlist toggle
🔎 Live Search by product name/brand
🧾 Cart Sidebar with total price calculation
🧭 Sticky Navbar on scroll
📱 Fully Responsive Layout
🎯 Clean and beginner-friendly code
🧱 Tech Stack
🌐 HTML5
🎨 CSS3 (Flexbox + Grid)
⚙️ Vanilla JavaScript (DOM, Events, Arrays)
